Using SSH In Putty On Windows


Written By Shaharukh Parmar on 4 Jan,2019


Overview


Putty ,is free and open source terminal emulator and network file tranfer application developed by Simon Tatham in January 8, 1999,

Download Putty from
https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html

Steps by Steps Using Guides

Steps:1

Open Putty


Step:2

Now enter IP Adrress of server or machine that you want to connect
For Instance
 Host or IP Address(Example.com or 145.123.65.49)
Port Number 22 Leave as it is(default)

Step:3

Now Click on Open button to start SSH Session

Step:4

If you first time connecting to server , you will see below Putty Security Warning


Step:5

Click on Yes Button to continue process other else process will be abruptly close

Step:6

Next Enter password of your choice
Note
you Will not see your cursor moving or any character typed(such as ****)

Step:7

Now you are successfully logged into your server with SSH
