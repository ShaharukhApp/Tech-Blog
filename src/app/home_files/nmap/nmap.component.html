Install NMAP On Windows


Written By Shaharukh Parmar on 7 Oct,2019


Overview

NMAP (Network Mapper) is free and open source network scanner developed by Gordon Lyon. NMAP is used to discover host and service on computer network by sending package and analyzing the responses , it provide additional feature to probe the computer network.

in this quick guide i will explain how to get installed NMAP on windows

Installing NMAP



Step:1
Go to Nmap download link and download stable version


or
- Go To Location where file is downloaded
- double click on .exe file to start installation(Run as Administrator)

Step:2

Now you see application is installing , accept licence agreement

Step:3

Now you can choose what component to Install, it would be good to install all the component



Step:4

Here you have to choose the install location , by default it install under Program Files(x86)\Nmap

Step:5

it will start installation process once done,you will get confirmation

Step:6

This marks Nmap is sucessfully installed
